Clio editor!

Clio is a functional lazy-evaluated programming language
targetting decentralized and distributed systems. It is made
to take advantage of multiple cpus and cpu cores (parallelism)
by default and to run on clusters and on the cloud easily. 
fn fib n:
  if n < 2:  n
      else: (n - 1 -> fib) +
            (n - 2 -> fib)

[0:10] -> * fib -> print
